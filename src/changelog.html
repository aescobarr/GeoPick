Version 2.0.0

  • Added new button Validate for validation of georeferences before sharing or exporting.

  • Added PostgreSQL back-end database, which can store georeferences in GeoJSON format for sharing via the applications API.
  • Added fields locationID and locality to the exported Darwin Core format.
